A machine learning project that detects whether a news article is real or fake.

# Tech Used
- Python
- Scikit-learn
- NLP (TF-IDF)
- Flask (optional)

# How it works
1. User enters news text
2. Text is cleaned and vectorized
3. Model predicts real/fake

# How to run
pip install -r requirements.txt
python main.py
